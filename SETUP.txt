========================================
  PetSitterPro - Setup Steps
========================================

Run these commands in order from your terminal.

----------------------------------------
STEP 1 - Start PostgreSQL
----------------------------------------
sudo service postgresql start


----------------------------------------
STEP 2 - Create the Database
----------------------------------------
sudo -u postgres psql -c "CREATE DATABASE pet_sitter_pro;"


----------------------------------------
STEP 3 - Go to the backend folder
----------------------------------------
cd /mnt/c/Users/Drech/Documents/computer_science/projects/pet_sitter_pro/backend


----------------------------------------
STEP 4 - Install dependencies
----------------------------------------
npm install


----------------------------------------
STEP 5 - Run the database migration
(This creates all your tables)
----------------------------------------
npx prisma migrate dev --name init


----------------------------------------
STEP 6 - Generate the Prisma client
----------------------------------------
npx prisma generate


----------------------------------------
STEP 7 - Start the dev server
----------------------------------------
npm run dev


----------------------------------------
DONE - Test it's working
----------------------------------------
Open your browser and go to:

  http://localhost:4000/health      <-- should say "database: connected"
  http://localhost:4000/graphql     <-- Apollo GraphQL playground
